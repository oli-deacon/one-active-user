Strava Insights is a useful reminder that personal software is often about arrangement rather than invention.

The underlying data already existed. The friction came from the shape it arrived in. By reworking that information into a more useful review pattern, the tool created clarity where a generic product created noise.

That kind of specificity is exactly the point.
