Kit Check demonstrates that personal software does not need to be technically ambitious to be meaningful.

Sometimes the value is simply that the tool understands the exact situation it is being used in. It knows what kind of trip this is, what changes with weather, and which omissions are costly enough to deserve their own prompts.

That is a small victory, but a real one.
